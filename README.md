# English
## "Kanban Board" Project
This project is a Kanban board that allows you to manage tasks in different states (To Do, In Progress, Done). It is developed using HTML, CSS, and TypeScript. This Kanban board enables you to create tasks, move them between columns, and delete them. It also allows you to add new columns to organize tasks according to your needs and delete them. It is designed to be simple and responsive.


# Français
## Projet "Kanban Board"
Ce projet est un tableau Kanban qui permet de gérer des tâches dans différents états (À faire, En cours, Terminé). Il est développé en HTML, CSS et TypeScript. Ce tableau Kanban permet de créer des tâches, de les déplacer entre les colonnes et de les supprimer. Il permet également d'ajouter de nouvelles colonnes pour organiser les tâches selon vos besoins ainsi que de les supprimer. Il est conçu pour être simple et responsive.
